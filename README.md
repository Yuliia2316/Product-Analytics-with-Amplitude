# Product Analytics with Amplitude

### Project Overview
In this project, I used **Amplitude** to analyze user journey and key product metrics for a mobile application. The main goal was to identify friction points in the conversion funnel and understand user retention patterns.

### Key Analysis performed:
* **Funnel Analysis:** Tracked the conversion from Main Page to Purchase. Identified the specific steps where the highest drop-off occurs.
* **Event Segmentation:** Analyzed the frequency of core app actions to identify "Power Users."
* **Retention Analysis:** Built N-day retention charts to measure how well the product keeps users over time.
* **User Composition:** Segmented the audience by platform, country, and version to find growth opportunities.

### Tools used:
* Amplitude, Behavioral Cohorts, Lifecycle Analysis.
